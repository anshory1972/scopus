Data & Methodology
  • Journal list: Scopus Source List May 2026, "Discontinued Titles" tab — 1,142 source IDs
  • Search query: AFFILCOUNTRY(Indonesia) AND SOURCE-ID({id1} OR … OR {id100}) AND PUBYEAR = {year} — batched in groups of 100
  • Doc types, OA, source types: targeted count queries via DOCTYPE(), OPENACCESS(), SRCTYPE() filters
  • Affiliations, authors, keywords, funding: sampled from up to 5,000 records using field= projection
  • Subject areas: derived from ASJC classification tags on discontinued source list
  • Benchmark countries: same 1,142 source IDs applied with AFFILCOUNTRY(Malaysia|Thailand|India|Vietnam)
  • API: Scopus Search API (Elsevier) · Fetched June 2026 · dev.elsevier.com
